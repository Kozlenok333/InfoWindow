# Описание
InfoWindow — это Python-скрипт для автоматического мониторинга активности окон пользователя. Он отслеживает смену активных приложений, записывает продолжительность работы в каждом окне и сохраняет данные в базу данных ClickHouse. Скрипт предоставляет детальную статистику использования программ для анализа продуктивности и тайм-менеджмента.

# Основные функции:

Автоматическое обнаружение смены активных окон

Измерение времени работы в каждом приложении

Сохранение данных в формате: дата, название окна, время использования

Интеграция с ClickHouse для хранения и анализа статистики

Работа в фоновом режиме с обработкой ошибок
